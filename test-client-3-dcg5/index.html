01 — What this stub covers

A stub that exercises every moving piece.

  • /01

    Slug generation and uniqueness

    The deploy MCP appends a 4-character random suffix and verifies no collision with existing directories before committing.

  • /02

    Atomic GitHub commit via Trees API

    HTML and any referenced assets land in a single commit on main. The repo stays clean and the deploy stays reproducible.

  • /03

    Vercel deployment polling

    The MCP polls until the new deployment reads READY, then returns the live URL. A timeout fallback ensures Claude still surfaces the URL even if polling stalls.

  • /04

    Plausible analytics injection

    Every deliverable carries the Plausible script in head, so views are tracked from the first visit forward. Open this URL in a private window and you should see it register.
