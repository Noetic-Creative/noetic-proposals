Copywriter Notes

Brand & Messaging Guidelines

✅

Approved language

→ "light-based treatment" → "light therapy" → "improve signs of dry eye" → "minimal discomfort" → "reduces dependency on eye drops" → "part of a treatment regime"
❌

Avoid

→ "IPL" alone as the product name → "treat symptoms" / "provide relief" → "zero pain" → "eliminate the need for drops" → Positioning as standalone cure → Text placed on device image
📌

Always include

→ OptiLight (O + L capitalised) → Location-specific clinic name → Disclaimer if clinical claims made → Goggles on patient in treatment photos → Handpiece on malar area only
